## 🚀 **How to Use**

### **1. Access the Dashboard**
```
Navigate to /admin or click "Admin Login" in the footer
```

### **2. Login**
- Email: Use Supabase Auth credentials
- First time? The system will initialize with sample data

### **3. Start Managing**
- Click any section in the sidebar
- Use "Add" buttons to create new content
- Click "Edit" to modify existing items
- Select multiple items for bulk actions

### **4. Upload Images**
- Any form with image upload has a file picker
- Click "Upload Image" button
- Select image file
- Preview appears automatically
- Image URL is saved to database

### **5. Manage Users** (Super Admin Only)
- Click "Users" in sidebar
- Add new admin users
- Change roles and permissions
- Reset passwords
- Bulk actions available
