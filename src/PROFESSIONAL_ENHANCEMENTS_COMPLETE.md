## 🔧 Customization Guide

### Update WhatsApp Number
```tsx
// File: /components/FloatingContact.tsx
// Line 10
href: 'https://wa.me/256XXXXXXXXX' // Replace with your number
```

### Update Phone Number
```tsx
// File: /components/FloatingContact.tsx
// Line 15
href: 'tel:+256XXXXXXXXX' // Replace with your number
```

### Adjust Carousel Speed
```tsx
// File: /components/Hero.tsx
// Line 46
}, 6000); // Change this value (milliseconds)
```

### Change Back to Top Trigger Distance
```tsx
// File: /components/BackToTop.tsx
// Line 9
if (window.scrollY > 400) // Change 400 to your preferred value
```
